
# AI Fitness Platform

A full-stack, AI-powered web application that dynamically generates personalized training and dietary protocols.

## 🚀 Live Demo
https://ai-fitness-platform-seven.vercel.app/

## 🛠️ Tech Stack
* **Frontend:** React, Vite, Tailwind CSS, React Router
* **Backend:** Node.js, Express.js
* **Database:** MongoDB Atlas, Mongoose
* **AI Integration:** Google Gemini LLM API
* **Authentication:** JWT (JSON Web Tokens), bcrypt

## ✨ Features
* Secure user authentication and authorization.
* Dynamic AI generation of diet and workout plans based on user metrics (height, weight, goals, experience level).
* Persistent data storage allowing users to save and retrieve their active protocols via a custom dashboard.
* Responsive, modern UI built with Tailwind CSS.
